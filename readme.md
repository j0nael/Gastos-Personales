# 💵 BienGastao

## 🧾 Una app encargada de manejar tus gastos sin problemas

**BienGastao** es una aplicación web diseñada para ayudarte a llevar un control claro y sencillo de tus finanzas personales.

Con BienGastao podrás:

* 🔐 Registrarte y gestionar tu cuenta de forma segura
* 💰 Registrar tus **ingresos** y **gastos** fácilmente
* 📊 Visualizar en qué estás gastando tu dinero
* 🏷️ Organizar tus gastos por **categorías**
* 🔍 Detectar esos pequeños “gastitos” que muchas veces pasan desapercibidos

## 🎯 Objetivo

El objetivo principal de BienGastao es ayudarte a tener una mejor conciencia financiera, permitiéndote identificar en que gastas tu dinero, para que lo uses en mejores gastos.

> 💡 *Pequeños gastos, grandes diferencias.*
